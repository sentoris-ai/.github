# Sentoris AI

**Control the Uncertainty.**

Sentoris is an open-source protocol and control plane that brings determinism, governance, and auditability to AI inference.

---

### ⚔️ The Challenge

AI inference is non-deterministic, making it difficult to:
- 🔁 Reproduce results for debugging or compliance
- 💰 Control costs in complex, multi-agent systems
- 📜 Audit decisions in regulated industries

### 🛡️ The Sentoris Solution

A governance layer that sits between your application and any LLM provider:
- **Trace**: Immutable, verifiable records of every inference
- **Replay**: Deterministic replay for regression testing
- **Diff**: Catch dangerous changes before they hit production
- **Budget**: Hard, real-time cost limits
- **Audit**: One-click compliance evidence
